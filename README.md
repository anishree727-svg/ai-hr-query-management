# AI HR Query Management System

## Overview
This project is an AI-powered HR query and leave management system built using Python and Streamlit.  
Employees can submit queries and requests, and the system provides responses using Generative AI.

## Features
- Employee login system
- HR query management
- Leave request handling
- AI-powered responses
- Simple Streamlit interface

## Tech Stack
- Python
- Streamlit
- SQLite
- OpenAI / Generative AI

## Installation

1. Clone the repository
git clone https://github.com/anishree727-svg/ai-hr-query-management

2. Install dependencies
pip install -r requirements.txt

3. Run the application
streamlit run app.py
